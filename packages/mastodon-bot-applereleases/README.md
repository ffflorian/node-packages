# mastodon-bot-applereleases [![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

Toots Apple software releases once they are published.

Currently deployed to [@applereleases@botsin.space](https://botsin.space/@applereleases).

## Prerequisites

- [Node.js](https://nodejs.org) >= 10.9
- [yarn](https://classic.yarnpkg.com)

## Installation

```
yarn
```

## Usage

```
yarn start
```
